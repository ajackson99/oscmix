URL Parameters Documentation

Currently supported devices:

- Fireface UCX II (Use Port 2)

- Fireface 802 (Use Port 2)

- Fireface UFX III (Use Port R or Port 4)

You can control the initial setup of the mixer by adding parameters to the URL. The base URL is:

Parameter Description Example
midi Set connection type to MIDI index.html?midi
midi-input Specify MIDI input device by name
(use quotes for names with spaces)		 index.html?midi-input="Fireface 802 (12345678) Port 2"
midi-output Specify MIDI output device by name
(use quotes for names with spaces)		 index.html?midi-output="Fireface 802 (12345678) Port 2"
auto-connect Automatically connect after setup
(requires midi parameter)		 index.html?midi&auto-connect=true
theme Set initial theme index.html?theme=style_tm_dark.css
